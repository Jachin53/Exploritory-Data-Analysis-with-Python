# 📊 Exploratory Data Analysis with Python – Resale Shop Dataset

## 📌 Overview
This project demonstrates **🔍 exploratory data analysis (EDA)** using Python on a resale shop dataset. The analysis covers:

- Data ingestion and cleaning  
- Statistical summaries  
- Feature engineering (date extraction, age classification, spending buckets)  
- Revenue analysis by demographic and product categories  

The goal is to uncover **business insights** such as customer segmentation, spending behavior, and revenue drivers with Python.
